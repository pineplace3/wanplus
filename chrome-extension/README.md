# ページキャプチャ＆PDF化ツール

Google Chrome拡張機能 - 画面をキャプチャしてPDF化するツール

## 機能

- 実行ボタンを押すと現在のページをキャプチャ
- **自動的に右カーソルキーを押下して次のページに遷移**
- ページ遷移後、6秒待機してから画面キャプチャを実行
- 遷移したページも自動的にキャプチャ
- 画面遷移がなくなったら自動終了
- キャプチャしたすべての画像を1つのPDFファイルに結合

## インストール方法

1. Chromeブラウザで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `chrome-extension` フォルダを選択
   - 場所: `c:\Users\willi\my-nextjs-app\chrome-extension`

## 使用方法

1. キャプチャしたい最初のページを開く
2. 拡張機能のアイコンをクリックしてポップアップを開く
3. 「キャプチャ開始」ボタンをクリック
4. **自動的に右カーソルキーが押下され、次のページに遷移します**
5. ページ遷移後、6秒待機してから自動的にページがキャプチャされます
6. 次のページに遷移できなくなったら自動終了
7. PDFファイルが自動的にダウンロードされます

**注意**: 手動で右カーソルキーを押すことも可能ですが、自動的に遷移するため通常は不要です。

## ファイル構成

- `manifest.json` - 拡張機能の設定ファイル
- `popup.html` - ポップアップUI
- `popup.js` - ポップアップのロジック
- `background.js` - バックグラウンドスクリプト（サービスワーカー）
- `content.js` - コンテンツスクリプト（ページと対話）

## 注意事項

- アイコンファイル（icon16.png, icon48.png, icon128.png）が必要です
- 次のページへのリンクが自動的に検出できない場合、自動遷移が失敗する可能性があります
- ページ遷移後、6秒間待機してからキャプチャを実行します（ページの完全な読み込みを待つため）
- PDF生成にはインターネット接続が必要です（jsPDFライブラリをCDNから読み込みます）
- 自動ナビゲーションは、ページ内の「次へ」リンクを自動的に検出してクリックします

## アイコンの作成

アイコンファイルが必要です。以下のサイズのPNG画像を作成してください：
- icon16.png (16x16ピクセル)
- icon48.png (48x48ピクセル)
- icon128.png (128x128ピクセル)

アイコンがない場合でも拡張機能は動作しますが、警告が表示される可能性があります。
